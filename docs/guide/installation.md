# 安装

## 环境要求

- Node.js 16+
- Vue 3.5+

## 包管理器

### npm

```bash
npm install midi-ui
```

### pnpm

```bash
pnpm add midi-ui
```

### yarn

```bash
yarn add midi-ui
```

## CDN

你也可以通过 CDN 的方式来使用 Midi UI：

```html
<!-- 引入样式 -->
<link rel="stylesheet" href="https://unpkg.com/midi-ui/dist/style.css" />

<!-- 引入组件库 -->
<script src="https://unpkg.com/vue@next"></script>
<script src="https://unpkg.com/midi-ui"></script>
```
